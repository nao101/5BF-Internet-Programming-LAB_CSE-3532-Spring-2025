Setting Up and Running the Django Website

Step 1: Activate the Virtual Environment (Windows Only)

In your root folder, open the Command Prompt and run the following command to activate the virtual environment:

.\venv\Scripts\activate.bat

Step 2: Migrations for the First Time

Before running the website for the first time, you'll need to set up the database. In the same Command Prompt, execute the following commands:

python manage.py makemigrations
python manage.py migrate

Step 3: Running the Development Server

Once the migrations are completed, you can run the server by using the following command:

python manage.py runserver

Step 4: Accessing the Website

After running the server, you can browse the website by visiting:

http://127.0.0.1:8000

Step 5: Creating an Admin Account

If you wish to access the admin panel, you need to create a superuser. In the Command Prompt, run the following command:

python manage.py createsuperuser

Fill in the required details with easy-to-remember credentials, and keep track of those credentials.

Step 6: Accessing the Admin Dashboard

Once the superuser is created, open your browser and go to:

http://127.0.0.1:8000/admin

Step 7: Adding Tour Packages

To add new tour packages, follow these steps:

  1. In the Admin Dashboard, locate the Tour Packages section.
  2. Click on Add New Package.
  3. Fill in the required details for the new package and hit Save.

Step 8: Exploring the Frontend

Feel free to explore the frontend and navigate around the website. Enjoy browsing, and if you need any assistance, feel free to reach out!
